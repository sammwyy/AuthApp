# AuthApp

Web based 2FA key manager to protect your accounts.

## Features

- Cloud storage.
- Group by namespaces.
- Key types:
  - TOTP (6 & 8 digits, 30 & 60 key intervals)
- Multiple algorithms:
  - SHA-1.
  - SHA-224.
  - SHA-256.
  - SHA-348.
  - SHA-512.
  - SHA3-224.
  - SHA3-256.
  - SHA3-348.
  - SHA3-512.
